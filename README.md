# 💰 Expense Tracker

Aplikasi pencatat pengeluaran & pemasukan harian yang simpel dan powerful.

## ✨ Fitur

- **📊 Dashboard** — Ringkasan pemasukan, pengeluaran, saldo harian & total
- **📷 Upload Struk** — Upload foto struk, parse otomatis total & kategori
- **📋 Riwayat** — Semua transaksi dikelompokkan per hari, filter by bulan/tipe
- **🏷️ Kategori** — Kelola kategori pengeluaran & pemasukan sendiri
- **💾 Local Storage** — Data tersimpan aman di browser

## 🚀 Cara Pakai

1. Buka `index.html` di browser
2. Klik tab **➕ Tambah Transaksi**
3. Upload struk → tulis hasil bacaan → klik "Parse & Isi Otomatis"
4. Atau isi form manual
5. Dashboard otomatis update!

## 📁 Struktur File

```
├── index.html   → Struktur halaman
├── style.css    → Tampilan dark-theme
└── app.js       → Logika & penyimpanan data
```

## 🛠️ Tech Stack

- HTML5 + CSS3 + Vanilla JavaScript
- LocalStorage untuk penyimpanan data
- 100% client-side, tidak butuh server!
